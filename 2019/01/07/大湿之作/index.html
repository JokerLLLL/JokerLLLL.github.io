大湿之作

前言

有幸结识一位大湿，也是突发文骚，向大师学习，而做略表敬意之至；
大湿笔名 顾南城 风雅的之名和其骚语性格判若两人，可谓隐其雅名苟于浊世。
达然情深意触之时，挥笔自成。今有幸阅达，秉承仰慕之意，刻之此文。

南 难成

文之大意，不言细说，南城兄流露之语。为之儿女情长之事，人之用情深处，言也无可表达尔。
人生 不易七八九，闲暇一二三。南兄初识其女也是青涩之时，言不善词，语不善达。其室友
曾为其想出各式表白，不曾帮于南兄。偶于平常餐食之际，吞吐而出 “我喜欢你”。此cp就在此
缘成。 春秋野草，花开花落。 几载时光，败于毕业季。 理性之人，不与事实雄辩。分手一年久，
未与之通电话。 圣诞有尔，闻于朋友圈 “离开你以后我才知道事实艰险” 。柴米油盐酱醋茶，
酸甜苦辣味道麻。南兄彻夜难眠。南城啊 南城啊 难成也 难成也！

吾于南兄

今借城南悲情之篇，顾有感此文，随笔法之杂乱，字迹之windows体，也算爽于性情。借此南兄，
执鼠标之Roll，操键盘之Rock。鄙人虽无笔名之称，但也想挥洒豪墨于之巅。
曾见南翔初中旧照，杀氏一族过而不及。现如今:江山易主，豪气冲云，飘逸发迹。但怎却来如此
掉发之所，于鄙人共事，实感悲怀。从南兄共事之两三晌，既然习得向日葵宝典，土味情话，舔狗不得
house等终生受用之书，实属鄙人幸事。今寒冬加班之际，粥铺畅谈，白粥，小菜。而吾曰：惜之 惜之，
不见那日令时小菜。 商贩答曰：每日两三碗，此更也无差所有咦。 南兄大话道： 而此尔等，怎有
剩乎于你我！ 我日(ri)： 碧莲不要否？尔食两碟，我食一碟，一可多于二乎？ 遂与之争辩。

粥过三巡，果腹于然。牛逼接一而二，实属猎米泰斗文豪之实。（不做产品可惜了）
时至送别鄙人，云水外，地铁边，雨水碧连天。赠与我怀情之作，至极荣幸。捧读而过地铁线，
非一两语而发之。

结语

夜以静深，南国的冬夜细雨长流，南兄的的情怀婉儿不露，素有情种之闺秀之称。
千禧又过19年，尔吾又临情人劫，同是单身若落入狗，（任尔东西南北风）
（ 相逢一笑名恩仇）（为啥你能帅过狗）（X X X X X X X ） 玉树临风看君尔。
