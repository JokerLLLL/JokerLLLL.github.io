centos7 yum搭建 lnmp

上次配的虚拟机没环境 今天yum搭建一下 lnmp环境

  1. 安装LNMP之前要安装EPEL，以便安装源以外的软件，如Nginx，phpMyAdmin等

        yum -y install epel-release
    yum -y install wget

  1. 安装Nginx

         yum -y install nginx 
     systemctl start nginx #启动nginx 
     systemctl enable nginx #设置开机启动

  2. 安装php

    直接安装php会是php5的版本，所以笔者安装之后又卸载了php5 ，如果想兼容安装，可以自行google。

           yum remove php* #删除之前版本
    # 设置php7的yum源 
    # 如果CentOS/RHEL 7.x:
    rpm -Uvh https://dl.fedoraproject.org/pub/epel/epel-release-latest-7.noarch.rpm
    rpm -Uvh https://mirror.webtatic.com/yum/el7/webtatic-release.rpm
    # 如果CentOS/RHEL 6.x:
    rpm -Uvh https://mirror.webtatic.com/yum/el6/latest.rpm
    #进行安装php7
    yum -y install php70w

    安装其他插件，这里应该要装的至少有php-fpm 如果一个个装嫌烦 ，就直接 yum -y install php70w* ,但好像有的会装不上去，必要时候 php -m 可以看下安装的模块有没有

    要运行PHP网页，要启动php-fpm解释器。
    修改php-fpm的配置
    vim /etc/php-fpm.d/www.conf
    找到以下两行，解除注释
    listen.owner = nobody
    listen.group = nobody
    找下以下两行，将各自的apache改为nginx
    user = apache -> user = nginx
    group = apache -> group = nginx
    然后开启fpm

     systemctl start php-fpm #开启php-fpm 
 systemctl enable php-fpm #开机自动启动
 ps -aux | grep php-fpm  #查看进程在不在

  1. mysql 安装

    直接命令行不好使，所以就加入源了。

      wget http://repo.mysql.com/mysql-community-release-el7-5.noarch.rpm
  rpm -ivh mysql-community-release-el7-5.noarch.rpm
  yum install mysql-server
  systemctl enable mysqld     #自启
  systemctl restart mysqld    #开启
   mysql -uroot
   mysql> set password for 'root'@'localhost' = password('123456');

  2. 安装phpmyadmin

     wget https://files.phpmyadmin.net/phpMyAdmin/4.4.12/phpMyAdmin-4.4.12-all-languages.zip 
 unzip phpMyAdmin-4.4.12-all-languages.zip 
 mv phpMyAdmin-4.4.12-all-languages /usr/share/nginx/html/phpmyadmin
 chown -R nginx.nginx /var/lib/php/session

  3. 之后就是配置nginx
    配置过程 就不说了，主要是配置好后，虚拟机能用，主机不能用 ，那把linux 的防火墙添加规则吧。

          /sbin/iptables -I INPUT -p tcp --dport 80 -j ACCEPT
